Common questions

Here are some of the most common questions that we get.
How can I upload images on the website? You can upload images by clicking on the 'Upload Images' button on the homepage.
Which regional languages are available on the website? The website supports regional languages such as English, Hindi, Punjabi, etc.
How will the uploaded images be used to fetch API results? The uploaded images will be processed to fetch relevant API results related to farming.
Can I view the API results on my smartphone? Yes, the website is responsive for smartphones, and you can view the API results on your smartphone.
What colors are used on the website related to farmers? The website uses colors inspired by nature and farming such as green, brown, and earth tones.
